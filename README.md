#  Montador de Móveis - Thiago Souza

Site institucional desenvolvido para divulgação de serviços de montagem de móveis em **Petrópolis - RJ**.

---

##  Sobre o projeto

Este projeto foi desenvolvido com foco em **conversão de clientes**, apresentando os serviços de forma clara, profissional e com contato direto via WhatsApp.

O objetivo é oferecer uma presença digital moderna para um profissional autônomo, aumentando sua visibilidade e geração de orçamentos.

---

##  Tecnologias utilizadas

* Vue 3
* TypeScript
* Vite
* Tailwind CSS

---

##  Funcionalidades

* ✅ Layout responsivo (mobile-first)
* ✅ Seções institucionais (Serviços, Sobre, Contato)
* ✅ Integração com WhatsApp
* ✅ Dark Mode (tema claro/escuro)
* ✅ Design moderno e otimizado
